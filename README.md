# Info-web-dev
Voici le rendu du rattrapage de web dev

Pour lancer le projet utiliser les commandes
./mvnw spring-boot:run depuis le répertoire task_planner, celui-ci lance le backend spring
cd ./Frontend
npm install pour installer les dépendances 
npm run dev pour lancer le frontend
